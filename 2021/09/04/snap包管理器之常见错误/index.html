snap包管理器之常见错误

使用 snap 安装软件时返回错误信息

输出的错误信息中包含如下关键字:

dial unix  /run/snapd.socket: connect: no such file or directory

解决方法是重启本地 snap 服务, 命令如下:

$: sudo systemctl restart snapd.service    //重启本地 snap 服务
